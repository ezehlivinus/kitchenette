# kitchenette
Host of Restaurants

# Usage
- User postman or insomnia to test the API

## Installation
- []: Clone the repository
- []: # Command: npm install
- []: set `.env` using the `.env.example` file
- []: # Command: npm install prisma -g
- []: # Command: npx prisma migrate dev
- []: # Command: prisma migrate
- []: # Command: npm run dev
